# Android AI Agent Skills

Reusable AI agent skills, workflows, standards, templates, and checklists for building production-grade Android applications.

## Repository Structure

```text
android-ai-agent-skills/
├── README.md
├── LICENSE
├── CONTRIBUTING.md
├── CHANGELOG.md
├── skills/
├── workflows/
├── standards/
├── templates/
├── checklists/
├── examples/
└── docs/
```

## Skills

Each folder under `skills/` contains a `SKILL.md` file focused on one Android engineering domain:

1. Product planning
2. Android architecture
3. XML UI design
4. Jetpack Compose
5. Navigation
6. Networking/API
7. Authentication
8. Local database
9. Firebase
10. Permissions
11. Media downloads
12. Security
13. Performance
14. Testing
15. Debugging
16. Refactoring
17. Code review
18. Play Store release

## How To Use

Copy or install the relevant `skills/<skill-name>/SKILL.md` into your AI agent workflow, then combine it with the matching files in `workflows/`, `standards/`, `templates/`, and `checklists/`.

Start with [docs/getting-started.md](docs/getting-started.md) and [docs/how-to-use-skills.md](docs/how-to-use-skills.md).
