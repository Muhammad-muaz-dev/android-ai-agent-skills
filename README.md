# Android AI Agent Skills

> A comprehensive collection of reusable AI agent skills, engineering standards, workflows, templates, and best practices for building professional, scalable, maintainable, secure, and production-ready Android applications.

## Overview

Developing modern Android applications requires consistent architectural decisions, coding standards, quality assurance, and engineering practices. While AI agents can significantly accelerate development, the quality of their output depends on the guidance they receive.

This repository provides a curated collection of reusable AI agent skills designed specifically for Android development. Each skill encapsulates domain knowledge, engineering standards, workflows, and decision-making processes that enable AI agents to produce high-quality, production-ready solutions consistently across projects.

Rather than relying on lengthy prompts for every task, these reusable skills establish standardized development practices that improve consistency, maintainability, and overall software quality.

- Produce scalable and maintainable applications

# What is an AI Agent Skill?

An AI agent skill is a reusable knowledge module that teaches an AI agent how to perform a specific engineering task according to predefined standards.

A skill may include:

- Development principles
- Architecture guidelines
- Coding standards
- Best practices
- Common pitfalls
- Decision-making rules
- Checklists
- Workflows
- Templates
- Examples

Each skill acts as a specialized expert that can be reused across multiple Android projects.
