# 18-play-store-release

Verify release builds, signing configuration, versionCode/versionName, ProGuard/R8 rules, privacy labels, permissions, SDK policy compliance, crash-free startup, billing/subscription behavior, and Play Console track readiness. Produce a release checklist and rollback plan before rollout.
